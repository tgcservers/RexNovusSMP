\section{Grundregeln}
\subsection{Stellung zu Kriegsverbrechen}
Folgende Kriegsverbrechen werden nicht als Verstoß gegen das Kriegsrecht erachtet:
\begin{enumerate}
    \item Völkermord
    \item Gezieltes Herbeiführen ziviler Opfer
    \item Angriffskrieg
    \item Misshandlung von Kriegsgefangenen
\end{enumerate}
